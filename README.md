# Anime Data Analysis 📊

This project analyzes an anime dataset using **Python and Pandas**.
The goal of this project is to explore anime data and extract useful insights such as episode counts, scores, and longest running anime.

---

## 📂 Project Files

* **anime.csv** → Dataset containing anime information
* **anime.ipynb** → Jupyter Notebook with the analysis code

---

## ⚙️ Tools Used

* Python
* Pandas
* NumPy
* Jupyter Notebook

---

## 🔎 Analysis Performed

The following analysis was performed on the dataset:

* Extracted **episode count** from the anime title
* Created a new column called **Episodes**
* Extracted **total running time**
* Identified **anime with the highest score**
* Found the **top 5 highest scoring anime**
* Determined **anime with the highest number of episodes**
* Identified the **longest running anime**
